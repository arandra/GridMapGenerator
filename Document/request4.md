# ScrollingCorridorModule 개선

1. 단계를 둬서 생성한다.
    1.1. 시작 점과 마지막까지의 코스를 만든다.
    1.2. 코스는 가능한 최소크기로 제작한다.
    1.3. 양쪽 끝에 이동 불가능한 타일이 늘어나거나 줄어들면서 코스를 보완한다.
    1.4. 코스를 기준으로 이동 가능한 타일은 좌우 대칭 크기로 생성한다. (예 y축이 10이고 x축으로 확장되는 맵의 경우, 0000011100으로 코스가 생성된 경우 0000111110이나 0001111111로 확장 될 수 있다. 0은 이동 불가능한 타일, 1은 이동 가능한 타일.)
    1.5. 코스의 각도 최대는 45도로 제한한다. (즉 이전 타일들과의 이동 가능여부 타일들의 차이는 좌, 우 1타일, 즉 2타일이 최대이다.)

2. wfc(wave function collaps)알고리즘을 이용해 타일을 확정한다.
