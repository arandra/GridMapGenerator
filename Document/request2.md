# Unity Editor 테스트 모드

* 어느 씬에서든 map 생성을 테스트 해보는 기능을 추가한다.
* 필요한 사항
 - 루트 오브젝트 지정
 - 각 타입별 매칭되는 타일 형식의 실제 어셋
* ui 의 "생성"버튼을 통해 특정 시드의 맵을 미리 보거나 랜덤한 시드의 맵을 확인할 수 있다.
* 사용자 시나리오
 - 사용자는 타일 타입을 추가한다.
 - 사용자는 타일 타입 별 이동 가능 여부를 설정한다.
 - 사용자느 타일 타입 별 대응되는 prefab을 등록한다.
 - "생성"버튼 클릭 시 루트 오브젝트의 모든 자식 오브젝트를 삭제하고 생성한다.
   - 루트 오브젝트에 자식 오브젝트가 있는 경우 경고 메시지를 출력한다.
   - 경고 메시지에 이 메시지를 다음 번에도 볼 것인지/생략할 것인지 체크하는 기능을 추가한다.
   - 경고에서 '취소'를 선택하면 자식 오브젝트를 삭제하지 말고 생성을 취소한다.
   - 경고에서 '확인'을 선택하면 자식 오브젝트를 삭제하고 생성한다.
 - 그리드 크기가 무한인 경우 미리 생성할 크기가 지정 되어야 한다.
   - 미리 생성할 크기는 런타임에서도 활용한다. 
 - 생성 시 지정된 프리펩에 대한 오브젝트 인스턴스를 각 타일 위치에 배치한다.
 - 시드를 사용자가 직접 입력하는 기능이 있고, 마지막 사용한 시드는 저장하고 unity를 재실행 하여도 복원되어야 한다.
 - "삭제"버튼을 추가하여 사용자가 생성(테스트)전의 환경으로 복원가능 해야 한다.
   - 복원이라는 표현은 루트의 모든 자식을 삭제하는 것을 의미하며, 그 외의 기능은 없다.
